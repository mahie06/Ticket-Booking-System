# Ticket Booking System

## Overview
A web-based Ticket Booking System developed using Java Servlets and MVC Architecture.

## Technologies Used
- Java
- HTML
- CSS
- JavaScript
- Java Servlets
- Apache Tomcat

## Features
- Ticket Booking
- Seat Selection
- Ticket Cancellation
- Booking Confirmation
- Seat Availability Tracking
- Input Validation

## Project Architecture
MVC (Model-View-Controller)

## Tools Used
- Eclipse IDE
- Apache Tomcat
- GitHub
